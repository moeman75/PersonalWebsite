Sundog Game!

Phaser JavaScript engine using the Iso-metric Plugin Rotate.

I've created an interactive Video Game for Sundog when i was working there. Using the Phaser JS Engine while implementing the isometric plug-in Rotate by lewster32. I was the lead designer and lead programmer for this entire project. I also created every sprite / image for this program using piskelapp.com.
